# Patient Appointment Management System (PAMS)

This is a simple CLI application for managing patients.

## Installation

1. Ensure Python 3.9+ is installed.
2. Install dependencies: `pip install -r requirements.txt`

## Running the App

Run `python main.py` and follow the menu options:
- Export patients to JSON file
- Print patients to console
- Exit

## Running Tests

Run `pytest tests/ -v` to execute the unit tests.
